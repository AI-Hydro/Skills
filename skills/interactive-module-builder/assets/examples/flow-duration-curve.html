1 · What is a flow-duration curve?

Rank every daily flow on record from highest to lowest, then ask: what fraction of days exceeded each value? Plot flow against that exceedance probability and you get the flow-duration curve (FDC) — a single picture of a river's entire flow regime, from floods on the left to droughts on the right.

Read it like this: the left tail is high flows you rarely see; the right tail is the low flow you can almost always count on (e.g. Q95, exceeded 95% of the time).

3 · Wet-and-steady vs flashy

The shape is the story. A flat FDC means steady, groundwater-fed flow; a steep FDC means a flashy basin that swings between floods and near-zero. Drag the wipe to compare two catchments with the same mean flow.

4 · Slope and water-supply reliability

A common index is the slope of the FDC between the 33rd and 66th percentiles — steeper means more variable. Adjust the basin's flashiness and re-run to see the slope and the Q95 low-flow guarantee change.

import numpy as np
import matplotlib as mpl, matplotlib.pyplot as plt
mpl.rcParams.update({'figure.facecolor':'#0a0a15','axes.facecolor':'#0f0f1e','text.color':'#e2e8f0',
    'axes.labelcolor':'#7dd3fc','xtick.color':'#94a3b8','ytick.color':'#94a3b8',
    'axes.edgecolor':(125/255,211/255,252/255,0.30),'grid.color':(125/255,211/255,252/255,0.10)})

rng = np.random.default_rng(11)
flash = {{flash}}                                  # ← bindParam: spread of the flow distribution
flows = 8.0*np.exp(flash*rng.standard_normal(3650))   # log-normal daily flows
fdc = np.sort(flows)[::-1]
p = 100*np.arange(1, fdc.size+1)/(fdc.size+1)         # exceedance %

def q(pct): return np.interp(pct, p, fdc)
slope = (np.log(q(33)) - np.log(q(66))) / (0.66 - 0.33)   # FDC slope index
q95 = q(95)

fig, ax = plt.subplots(figsize=(11, 4))
ax.semilogy(p, fdc, color='#00DDFF', lw=2)
ax.axvline(95, color='#34d399', ls='--', lw=1); ax.scatter([95],[q95], color='#34d399', zorder=5)
ax.set_xlabel('Exceedance probability (%)'); ax.set_ylabel('Q (m³/s, log)')
ax.set_title(f'FDC slope index = {slope:.2f}    Q95 = {q95:.2f} m³/s', color='#e2e8f0')
ax.grid(True, which='both', alpha=0.3); plt.tight_layout()

6 · Summary

  • The FDC plots flow against exceedance probability — the whole flow regime in one curve.
  • A flat curve = steady, groundwater-fed; a steep curve = flashy and rain-driven.
  • The low-flow tail (Q90–Q95) defines dependable supply; the FDC slope index quantifies variability.
