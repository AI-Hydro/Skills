1 · Where does water collect?

Stand anywhere in a catchment and two things decide how wet that spot tends to be: how much land drains into it from upslope, and how steep it is (steep ground sheds water, flat ground holds it). The Topographic Wetness Index (TWI) turns those two terrain facts into a single map of likely saturation — built from nothing but a digital elevation model.

Intuition: valleys with large upslope areas and gentle slopes light up as wet; steep ridges stay dry.

3 · The TWI equation, built up

The index is compact: the log of upslope area divided by the local slope. This short animation builds it term by term.

5 · Compute it yourself

The same calculation in NumPy: build a DEM, estimate slope, approximate upslope area, and map ln(a/tanβ). Run the cell to render the wetness map.

python Topographic Wetness Index from a synthetic DEM 
import numpy as np
import matplotlib as mpl, matplotlib.pyplot as plt
mpl.rcParams.update({'figure.facecolor':'#0a0a15','axes.facecolor':'#0f0f1e','text.color':'#e2e8f0',
    'axes.labelcolor':'#7dd3fc','xtick.color':'#94a3b8','ytick.color':'#94a3b8'})

# Synthetic ridge-and-valley DEM (metres)
n = 80
y, x = np.mgrid[0:n, 0:n]
dem = 60*np.exp(-((x-40)**2)/600) + 0.35*y + 3*np.sin(x/6)   # central valley + regional tilt

# Slope from gradients (cellsize = 30 m)
gy, gx = np.gradient(dem, 30.0)
slope = np.sqrt(gx**2 + gy**2)
tan_beta = np.clip(slope, 0.001, None)

# Upslope area proxy: lower, flatter cells accumulate more drainage
acc = (dem.max() - dem) + 1.0
a = acc / (tan_beta + 0.05)

twi = np.log(a / tan_beta)

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 5))
im1 = ax1.imshow(dem, cmap='cividis'); ax1.set_title('Elevation (m)', color='#e2e8f0'); ax1.axis('off')
im2 = ax2.imshow(twi, cmap='YlGnBu');  ax2.set_title('TWI  (bright = wet)', color='#e2e8f0'); ax2.axis('off')
fig.colorbar(im1, ax=ax1, fraction=0.046); fig.colorbar(im2, ax=ax2, fraction=0.046)
plt.tight_layout()

7 · Summary

  • TWI = ln(a / tanβ): big upslope area and gentle slope mean a wet cell.
  • It is derived entirely from a DEM — no soil or rainfall data required.
  • TWI maps highlight valley floors and convergence zones, useful for siting wetlands, riparian buffers and saturation-prone areas.
